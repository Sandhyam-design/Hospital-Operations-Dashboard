# 🏥 Hospital Operations Analytics Dashboard

An interactive Business Intelligence dashboard developed using **SQL** and **Power BI** to monitor hospital performance, improve operational efficiency, and support data-driven healthcare decision-making.

---

# 📌 Project Overview

Healthcare organizations generate large volumes of operational and financial data every day. This project transforms raw hospital data into meaningful business insights through interactive dashboards that help management monitor key performance indicators (KPIs), optimize resource utilization, and improve patient care.

---

# 🎯 Business Problem

Hospital management needs timely and accurate information to make operational decisions. Without a centralized reporting system, it becomes difficult to monitor admissions, occupancy, revenue, departmental performance, and patient outcomes.

The objective of this project was to develop an interactive dashboard that enables management to:

* Monitor hospital performance in real time
* Track patient admissions and readmissions
* Analyze revenue trends
* Evaluate departmental efficiency
* Improve operational planning through data-driven insights
