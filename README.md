# Hospital-Operations-Dashboard
Hospital Operations Analytics Dashboard built using SQL and Power BI to monitor hospital KPIs, operational efficiency, and business performance.

**Business Problem**

Healthcare organizations manage large volumes of operational and financial data across multiple departments. The objective of this project was to consolidate hospital data into an interactive dashboard that enables management to monitor admissions, patient flow, occupancy, revenue, and departmental performance while identifying operational trends and areas for improvement.

**My Responsibilities**

**Key Performance Indicators (KPIs)**

Total Patient Admissions
Patient Readmissions
Hospital Revenue
Occupancy Rate
Average Length of Stay (ALOS)
Department Performance
Bed Utilization
Treatment Cost Analysis

**Business Insights**

Identified departments with consistently higher patient readmission rates.
Analyzed occupancy trends to highlight periods of peak resource utilization.
Evaluated revenue contribution by department and treatment category.
Monitored patient admission patterns to support operational planning.
Improved management visibility into hospital performance through interactive KPI dashboards.
